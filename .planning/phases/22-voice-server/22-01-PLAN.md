<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser -> Pipecat WS (7860) | Raw audio frames from browser microphone |
| Next.js -> Python health (7861) | Internal HTTP proxy, localhost only |
| Python -> SQLite | Cross-process writes to shared DB |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-22-01 | Spoofing | WebSocket port 7860 | accept | Single-user local tool; no auth needed per PROJECT.md. Listens on 0.0.0.0 but behind Cloudflare tunnel which only exposes Next.js port 3002 |
| T-22-02 | Tampering | SQLite writes from Python | mitigate | WAL mode + busy_timeout=5000 on every connection; INSERT OR IGNORE prevents duplicate request_ids |
| T-22-03 | Information Disclosure | API keys in .env | mitigate | .env.example has no real values; .env is gitignored; keys never sent to client |
| T-22-04 | Denial of Service | Single-client WS limit | accept | WebsocketServerTransport enforces one client; second connection closes first. Documented behavior, not a bug |
| T-22-05 | Elevation of Privilege | Python subprocess (say) | mitigate | subprocess.run with explicit command list (no shell=True); only called as absolute last resort TTS fallback |
</threat_model>
