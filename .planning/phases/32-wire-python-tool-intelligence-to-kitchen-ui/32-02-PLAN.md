# Phase 32 Plan 02: Types + Algorithm Port (Wave 1)

## Goal

Add TypeScript types for the context pack and similar-task response, then port the
Python `contextMatchSignal` algorithm into `apps/kitchen/src/lib/tool-attention.ts`
and expose a `getSimilarTaskRecommendations(context, limit)` function.

## Key Algorithm Details (do not deviate)

The `contextMatchSignal` function MUST use these exact multipliers:
- `task_type` match: `count * 2`
- `repo` match: `count * 2`
- `agent_id` match: `count * 1`
- `tags` match: `count * 1` (per tag)

Overall score formula:
```
overallScore = textMatchScore * 10 + outcomeSummary.score + contextMatchScore * 3
```

Privacy constraint: NEVER read the `task` field from JSONL records — only `metadata.*` fields.

2. Add constant sets matching Python:
   ```typescript
   const SUCCESS_OUTCOMES = new Set(["helped", "success", "successful", "useful", "pass", "passed", "worked"]);
   const FAILURE_OUTCOMES = new Set(["failed", "failure", "not_helpful", "not helpful", "miss", "error", "blocked"]);
   ```

3. Implement `contextMatchSignal(outcome: ToolAttentionOutcome, context: ToolAttentionContextPack): number`:
   - Read ONLY `outcome.metadata.*` — never `outcome.task`
   - `task_type` match: if `context.task_type && outcome.metadata?.task_type === context.task_type` → +2
   - `repo` match: if `context.repo && outcome.metadata?.repo === context.repo` → +2
   - `agent_id` match: if `context.agent_id && outcome.metadata?.agent_id === context.agent_id` → +1
   - `tags` match: for each tag in `context.tags ?? []`: if `(outcome.metadata?.tags as string[] ?? []).includes(tag)` → +1
   - Return the total

4. Implement `getSimilarTaskRecommendations(context: ToolAttentionContextPack, limit = 10): SimilarTaskResponse`:
   - Call `getToolAttention("", 100)` to get all capabilities and outcomes
   - For each capability, aggregate contextMatchScore across all outcomes matching its toolId
   - Compute `overallScore = 0 * 10 + (outcomeSummary?.score ?? 0) + contextMatchScore * 3`
     (textMatchScore is 0 because query is "")
   - Filter to capabilities where `contextMatchScore > 0 || (outcomeSummary?.score ?? 0) > 0`
   - Sort descending by overallScore
   - Map to `SimilarTaskRecommendation[]` with a human reason string
   - Return `{ context, recommendations: sliced, timestamp: now() }`
